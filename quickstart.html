CSC DCAF Quick Start Guides¶

The CSC Data Center Automation Framework (DCAF) is a collection of resources designed to automate various platforms and deployments within the data center. It is comprised of CSC public open source projects, and various community based open source projects.

The series of guides listed below provides the steps to take to start using CSC DCAF relatively quickly. See the requirements section in each guide to ensure you have the correct setup to perform the steps as they’re provided.

  • Autodeploynode Quick Start Guide
  • Bare-metal-OS Quick Start Guide
  • RHEL-OSP Quick Start Guide
  • KVM-Host Quick Start Guide
  • Slimer Quick Start Guide
  • Ansible-ScaleIO Quick Start Guide
